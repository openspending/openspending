Create a dataset

Hi there! If you have a spending or budget dataset that you can contribute, please give some basic information here.

In the next steps, you'll
  1. add references to data sources,
  2. create a data model to describe the format,
  3. test the correctness of your model,
  4. import the data,
  5. and generate visualized views.
The identifier will become part of the dataset URL and cannot be changed later. It may only contain alphanumeric characters, dashes and underscores.
